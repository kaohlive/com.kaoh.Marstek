⚠️ This is a debugging tool for advanced users. Writing to unknown registers may cause unexpected behavior.

1. Select Device

2. Manual Register Access

Enter a decimal register address (e.g., 43000 for User Work Mode)

Enter a decimal value to write (0-65535)

3. Device State Dump

Poll all known registers and display their raw values for debugging.

4. Work-mode Event Buffer (diagnostic)

Passive ringbuffer of work-mode write/read events. Use this to measure how many polls (and ms) it takes for the device to expose a freshly written work_mode value via reads. No behaviour change — purely logging.
