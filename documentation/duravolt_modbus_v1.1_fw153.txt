Marstek Duravolt / Venus D - Modbus Register Reference
======================================================================

Source: Community datasheet extending the official Duravolt PDF
        'Duravolt Plug-In thuisbatterij 485 protocol v1.1'
        Source PDF: https://duravolt.nl/wp-content/uploads/Duravolt-Plug-in-Battery-Modbus.pdf
        Datasheet checked: 2025-08-14
        Scanned on: firmware v153, BMS v213

Protocol:
  Standard Modbus RTU, default address 1
  Serial: 115200 baud, 8 bits, no parity, 1 stop bit

QUIRK: If a register contains more than 2 bytes of data, it MUST be
       read in one function call (count = full byte width / 2).
       Reading consecutive registers in separate calls returns
       invalid data.

Scans are made on firmware v153 and BMS v213.
Protocol specification:
Standard Modbus RTU Protocol
Modbus default Address: 1
baud:115200, 8bit, No parity bit, 1bit Stopbit
Quirk: If register contains more than 2 bytes of data, data should be read in one function call instead of consecutive calls to consecutive registers or it will be invalid. Therefore if any data is present, one can deduct where a range must be split. If data is zeroes, the range is left and appended with a ‘?’.
