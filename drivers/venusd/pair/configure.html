Marstek Venus D Connection

This driver is for the Marstek Venus D hybrid battery with built-in PV inverter. Some resellers market this device as "Duravolt" — same hardware, same driver. If you have a Venus E or V3 battery, please use the "Venus Battery" driver instead — Venus D has a different Modbus register layout and the drivers are not interchangeable.

Venus D officially speaks Modbus RTU over RS485 baud 115200. To connect it to Homey over the network you can either use a Modbus TCP-to-RS485 gateway (EW11, Waveshare) or, on recent firmware, the Venus D's built-in Ethernet port if it supports native Modbus TCP.
